# <div align="center">delimiter - 是否打印分割符</div> #

&lt;delimiter&gt;
<pre>
是否打印分割符
</pre>

#### 说明 ####

<pre>
在&lt;for&gt;中，设置是否打印当前循环的分隔符，
必须在&lt;for&gt;语句块中，并且for中有指定分隔符delimiter的值。
</pre>

#### 属性 ####

<pre>
<b>run（必须）</b>
for代码块内，本次循环是否输出分隔符

<b>属性</b>
true:当前这次循环运行分隔符（默认）
false:当前这次循环不运行分隔符
</pre>
